## Webinar Outline

### Opening (5 min) — The Hook

### Part 1 (10 min) — The 5 Silent Killers of AI Apps
1. **Expired API Keys** — The most common. API providers rotate keys; no-code apps rarely auto-update
2. **Rate Limit Shifts** — App grows → more API calls → 429 errors → degraded UX (no code change needed)
3. **Schema Drift** — API providers update response formats → your data pipeline silently breaks
4. **Broken Webhooks** — Deployments change endpoints; webhooks 404 silently
5. **Error Spikes** — A dependency update introduces new errors without breaking the app

### Part 2 (10 min) — Live Debug: Watch Us Catch a Real Failure
- Screen share: Nightlamp dashboard

- Walk through the playbook: symptom → diagnose → fix → prevent

- Week 1: Audit all API keys and webhooks (template provided)
