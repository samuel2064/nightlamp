### How It Works (3 steps)
1. **Connect (2 min)** — Link your app. We scan every API key, webhook, and integration.
2. **We find what's broken (90 seconds)** — Our engine detects expired keys, broken webhooks, rate limit shifts, and schema drift.
3. **We fix it (or tell you how)** — Auto-fix for 80% of failures. For the rest, step-by-step playbooks.

| Failure | What Happens | Nightlamp |
|---------|-------------|-----------|
| Expired API keys | 401 errors. Feature stops working. | Detects + fixes in 12 min |
| Rate limit shifts | 429 errors. Degraded UX. | Alerts before users notice |
| Schema drift | Data pipeline breaks silently | Detects mismatch in real-time |
| Broken webhooks | Data loss. No one checks. | Verifies endpoints daily |
| Error spikes | Slower app. Churn grows. | Alerts + playbook match |
